# AI Autism Therapy System

## Project Overview
This project uses Machine Learning techniques to analyze autism therapy session data and predict therapy success outcomes.

The system processes behavioral and session-related features such as:
- Session duration
- Interaction count
- Navigation patterns
- Completion time
- Errors
- Overall score

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## Machine Learning Models
The project compares multiple classification algorithms including:
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)
- Decision Tree
- Logistic Regression
- Random Forest
- Naive Bayes
- Perceptron

---

## Data Preprocessing
The following preprocessing steps were applied:
- Missing value checking
- Feature encoding
- Feature scaling
- Oversampling & undersampling
- Feature importance analysis

---

## Model Performance

- Accuracy: 97.22%
- Precision: 100%
- Recall: 96.3%
- F1-score: 98.11%
